## 📌 About Ambitions
**Ambitions** is a fully custom FiveM framework designed for **performance, scalability, and security**. It is built to handle **high concurrency** efficiently, making it suitable for **both small and large servers**. 

With an **optimized structure**, **modern coding standards**, and a **developer-friendly approach**, Ambitions aims to be a **fast, flexible, and highly secure** framework that stands out from traditional solutions.

## ⚡ Key Features
- 🔹 **Highly Optimized** – Built for high player concurrency with minimal resource usage
- 🔹 **Secure & Lightweight** – Only necessary operations run client-side to ensure security
- 🔹 **Scalable** – Works efficiently for both small RP servers and large-scale communities
- 🔹 **Developer-Friendly** – Easy to understand, well-structured, and highly customizable
- 🔹 **MariaDB Support** – Optimized for MariaDB (⚠️ **No XAMPP support** – See below)
- 🔹 **Built-in Localhost Setup** – Includes core components for easy local development
- 🔹 **Community Contributions Welcome** – PRs, issues, and feature requests are encouraged!

## 🚀 Installation & Requirements
To run Ambitions, ensure your environment meets the following requirements:

### 🔧 **System Requirements**
- **FiveM Server Artifact:** Must be **12208 or higher**
- **Database:** **MariaDB** (⚠️ **DO NOT use XAMPP!** See explanation below)
- **Database Management:** **HeidiSQL** is highly recommended

### ❌ **Why Not XAMPP?**
XAMPP runs an **outdated version of MariaDB/MySQL**, which can cause:
- ⚠️ **Performance issues**
- ⚠️ **Database inconsistencies**
- ⚠️ **Security vulnerabilities**
- ⚠️ **Lack of compatibility with modern FiveM setups**

To avoid these issues, **use MariaDB directly with HeidiSQL** for the best stability.
